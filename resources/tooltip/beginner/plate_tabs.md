#### Plate Tabs

**What it does:** Each plate tab represents a separate build plate, letting you organize multiple print jobs within a single BambuStudio project.

**Why it matters:** Instead of opening separate project files for each print, you can set up several plates with different models or settings and send them to the printer one at a time, keeping everything organized in one place.

**Typical values:** Not applicable -- click the "+" tab to add a new plate, or click an existing tab to switch between plates.

**Tip:** Use multiple plates to group parts by material or priority. For example, put urgent parts on Plate 1 and optional extras on Plate 2 so you can print the important ones first.
