#### 3D Editor Tab

**What it does:** Opens the main workspace where you import, position, rotate, scale, and arrange your 3D models on the virtual build plate.

**Why it matters:** This is where all print preparation begins. You need to place your model correctly on the plate and adjust its size and orientation before configuring any print settings.

**Typical values:** Not applicable -- this is the default tab when you open BambuStudio.

**Tip:** Use the toolbar on the left to switch between Move, Rotate, and Scale tools. Right-clicking a model reveals more options like placing it flat on a specific face.
