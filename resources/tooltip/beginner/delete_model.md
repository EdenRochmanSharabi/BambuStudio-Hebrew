#### Delete Model

**What it does:** Removes the currently selected model from the build plate so it will not be included in the print.

**Why it matters:** If you loaded the wrong file, want to start over, or need to clear space on the plate for a different arrangement, deleting a model is the quickest way to clean up.

**Typical values:** Not applicable -- select the model and press Delete or use the right-click menu.

**Tip:** If you accidentally delete a model, use Ctrl+Z (Cmd+Z on Mac) to undo immediately and bring it back with all its settings intact.
