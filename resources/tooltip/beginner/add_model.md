#### Add Model

**What it does:** Imports a 3D file (such as STL, 3MF, or OBJ) into BambuStudio and places it on the virtual build plate.

**Why it matters:** This is how you get your 3D model into the slicer so you can configure settings and prepare it for printing. Without adding a model, there is nothing to slice.

**Typical values:** Supported formats include STL (most common), 3MF (preserves settings and colors), OBJ, and STEP.

**Tip:** 3MF files are preferred when available because they can carry print settings, color information, and support painting -- saving you setup time.
